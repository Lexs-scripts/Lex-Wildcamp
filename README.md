# 📖 Overview

**Jay-Wildcamp** is a premium and fully optimized camping system built for **RedM roleplay servers**.

Designed to deliver a realistic and immersive outdoor experience, the system allows players to create, customize, relocate, and manage fully synchronized camps across the server world.

From advanced prop placement mechanics to wagon-based camp transportation, Jay-Wildcamp combines performance, flexibility, and roleplay immersion into one complete package.

Built with scalability and server performance in mind, the resource supports multiple frameworks and includes extensive configuration options for every server style.

---

# ✨ Features

## 🏕️ Advanced Camp Management
- Place, install, move, and remove camp props
- Fully synchronized camp states between all players
- Smooth and immersive edit mode system
- OneSync optimized for large servers

---

## 🛒 In-Game Camping Store
- Interactive in-game store system
- 3D prop preview with automatic rotation
- Category filtering & live search system
- Shopping cart with quick buy support
- Weight validation system

---

## 🔨 Realistic Placement System
- Full placement controls:
  - Move
  - Rotate
  - Raise / Lower
  - Snap to ground
- Immersive hammer installation animation
- Dynamic prop interaction system

---

## 🚚 Camp Relocation System
- Pack your entire camp into a wagon
- Move and relocate camps anywhere allowed
- Persistent camp restoration after deployment

---

## 👥 Member & Permission System
- Invite camp members
- Granular permission management
- Individual permissions for:
  - Storage access
  - Menu access
  - Place/remove props
  - Move camp
  - Manage members

---

## 🌎 Server Systems
- Town restriction system
- Camp radius enforcement
- Camp blip visibility toggle
- Server-wide favorites & voting panel
- Persistent database & JSON storage

---

## 📚 Tutorial & User Experience
- Built-in tutorial system
- Step-by-step onboarding
- GIF & image support for tutorials
- Custom notification integration

---

## ⚙️ Compatibility
- ✅ VORP Framework
- ✅ RSG Framework
- ✅ Multi-language Support
  - English
  - Turkish
  - German
